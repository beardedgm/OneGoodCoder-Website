# OneGoodCoder — Derrick Dedmon

Personal portfolio website for **Derrick Dedmon**, Director of Communications & Creative Strategist with 25+ years of experience in web development, brand management, digital strategy, and communications leadership.

**Live site:** [onegoodcoder.com](https://onegoodcoder.com)

## Tech Stack

- **HTML5** — semantic markup with accessibility best practices
- **CSS3** — custom properties, CSS Grid, Flexbox, scroll-triggered animations
- **Vanilla JS** — Intersection Observer animations, smooth scroll, modals
- **Google Fonts** — Syne + DM Sans
- **Formspree** — contact form handling
- **Google Analytics** — site analytics via gtag.js

## Structure

```
├── index.html          # Single-page site
├── style.css           # All styles
├── main.js             # Interactions and animations
├── robots.txt          # Crawler directives
├── sitemap.xml         # XML sitemap
└── images/             # Portfolio and profile images
```

## Local Development

Serve the static files with any HTTP server:

```bash
python -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).
